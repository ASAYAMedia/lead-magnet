Part 1: Check Your Hardware

Minimum Requirements

  • RAM: 8GB (16GB recommended)
  • Storage: 20GB free space
  • GPU: Optional but recommended (8GB+ VRAM)
  • OS: Windows 10/11, macOS, or Linux

Quick Check (Windows)

  1. Press Win + R, type dxdiag, press Enter
  2. Look at "Display" tab for your GPU
  3. If you see NVIDIA RTX 2060 or better, you're golden
  4. If no discrete GPU, you can still run on CPU (slower)

Part 2: Install Ollama

Step 1: Download

Go to: https://ollama.com

Click the download button for your OS.

Step 2: Install

  • Windows: Run the installer, click through
  • Mac: Drag to Applications
  • Linux: Open terminal, run:
curl -fsSL https://ollama.com/install.sh | sh

Step 3: Verify

ollama --version

You should see a version number. If not, restart your computer.

Part 3: Download Your First Model

Model Size Guide:
3B-8B = Fast, good for most tasks
13B-20B = Smarter, need 16GB+ RAM
70B = GPT-4 level, need powerful GPU

For your first model, use Llama 3.1 (8B):

ollama pull llama3.1:8b

This downloads ~5GB. Wait for completion.

Part 4: Start Chatting

Terminal Method (Quick Test)

ollama run llama3.1:8b

Type: Hello, are you running locally?

Type /bye to exit.

Web Interface (Better)

CommandWhat It Does

Part 7: Common Problems

"CUDA out of memory"

Fix: Use smaller models (3B-7B) or run on CPU

"Model not found"

Fix: Run ollama pull modelname first

Slow responses

Fix: Close other apps, use smaller models, upgrade RAM
